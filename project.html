Integrated Laboratory Management Information System (ILMIS)

Website (Proyek Terbaru)

Sebuah sistem manajemen laboratorium berbasis PHP (CodeIgniter 4) yang dikembangkan untuk mengelola jadwal, inventaris, dan autentikasi SSO laboratorium. Sistem ini merupakan pengembangan dari SITARIS dengan fitur lebih lengkap dan performa lebih baik.

  • Peran: Full Stack Developer
  • Teknologi: CodeIgniter 4, PHP 8, MariaDB, HTML, CSS, JavaScript
